# 2026 주요정보통신기반시설 기술적 취약점 분석·평가 자동화(LINUX)
WIP

## 개요
KISA 2026 가이드를 기준으로 LINUX RHEL9 설치 기본값 서버를 자동화 스크립트로 진단합니다.
이에 엄격히 부합하는 스크립트를 작성 및 시행함으로서 표준 준수 및 안전한 서버 운영을 도모합니다.

## 시나리오
취업 준비생 K군은 리눅스 서버 공부중 우연히 KISA의 가이드를 발견하게 되었다.
K군 본인의 서버가 기본 환경에서 운영됨을 알고, 보다 안전한 운영을 위해 가이드 표준을 준수하기로 한다.
스스로 검사하고 조치를 취하고자 하였으나, 사항이 많아 자동화로 진행하고자 Bash 스크립트를 작성하기로 마음먹었다.
